# EEG/ECG Examination Neural Networks

- Locally trained Neural Network project

- For my ECG model, I used the PTB-XL dataset from https://physionet.org/content/ptb-xl/1.0.3/
- Will use the dataset here for my EEG model https://www.kaggle.com/datasets/buraktaci/mtouh?resource=download

- After training, we can use the .pth files for inference
  - Just input some data, wait for the model to run, look at the graphs and the text we get as a result.
